# etl-azure-snowflake-hybrid
Modular ETL pipeline for Azure Blob → Snowflake TGT and Snowflake SRC → Snowflake TGT ingestion with L1 (STG)→ L2 (MAIN), automatic logging, row-count validation, multithreading, and Slack-based monitoring. Supports parallel table ingestion that is fully scalable, configurable environment-based execution, fully open-source adaptable architecture.
